# LLMGuard-QA

LLMGuard-QA is an AI-powered web application designed to evaluate and test Large Language Model (LLM) responses. The system helps analyze AI-generated outputs based on quality, correctness, and safety through an easy-to-use interface.

## Features

- AI response evaluation
- User-friendly frontend
- Secure backend APIs
- Testing module for validating responses
- Clean and scalable project structure

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python (FastAPI)
- Database: MySQL
- AI/ML: Python
- Version Control: Git & GitHub

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/LLMGuard-QA.git
```

2. Navigate to the project folder

```bash
cd LLMGuard-QA
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the backend

```bash
python app.py
```
